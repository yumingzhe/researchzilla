Researchzilla
=============

在大部分高校中各个学科、专业都设立了研究室，学校中的科研人员、教育工作者等几乎都隶属于某个研究室。但研究室内部发布通知、论文、会议记录等文档时大都以纸质文件形式或口头方式传播，在这种情况下，信息传播速度有限，甚至会延误内部成员间的交流与合作，极大地降低了工作效率。因此，为研究室建立一个统一的集信息发布、组内协作交流的平台以供内部成员使用，是研究室的重要组成部分，也是研究室向信息化发展的必经之路。架设研究室网站好处诸多，首先，研究室网站可以作为该研究室或研究机构的官方站点，可供外界访问，了解该机构最新的研究成果、项目进度等信息，从而宣传其科研能力。其次，对于学生群体，可以通过访问不同的研究室网站了解自己所喜爱的科研方向，从而选择相应的导师，便于研究室培养更多的学生。最后，网站最重要的功能在于给内部成员提供了一个信息平台，成员可以获取该研究室内部的最新动向，可以相互交流与合作，让成员更专注于科研而非获取信息的过程。因此，本项目主要研究如何设计并实现研究室网站，从而为研究室提供一种可快速架设网站的解决方案。
